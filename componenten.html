TILTSHIFT Design System

Een complete component library voor digitale transformatie projecten

Instructies voor gebruik

  • CSS Files: Include de vier CSS bestanden in de juiste volgorde: tiltshift-base.css, tiltshift-tokens.css, tiltshift-utilities.css, tiltshift-components.css
  • Font Support: AtkinsonHyperlegible en HelveticaNeue fonts zijn inbegrepen voor toegankelijkheid
  • BEM Methodology: Alle componenten gebruiken Block Element Modifier naming convention
  • Design Tokens: Gebruik CSS custom properties (--ts-*) voor consistente styling
  • Responsive: Alle componenten zijn mobiel-eerst en volledig responsive
  • Utility Classes: Gebruik utility classes voor spacing (mt-1, mb-2), layout (flex, grid) en colors

Implementatie

CSS Bestanden Includen

Voeg deze CSS bestanden toe aan je HTML in de juiste volgorde:

<!-- TILTSHIFT Design System --> <link rel="stylesheet" href="https://cdn.tiltshift.nl/tiltshift-base.css"> <link rel="stylesheet" href="https://cdn.tiltshift.nl/tiltshift-tokens.css"> <link rel="stylesheet" href="https://cdn.tiltshift.nl/tiltshift-utilities.css"> <link rel="stylesheet" href="https://cdn.tiltshift.nl/tiltshift-components.css">

Design Tokens Gebruiken

Gebruik CSS custom properties voor consistente styling:

/* Kleuren */ color: var(--ts-color-primary); background-color: var(--ts-color-primary-dark); /* Spacing */ margin: var(--ts-space-3); padding: var(--ts-space-2) var(--ts-space-4); /* Typography */ font-family: var(--ts-font-primary); font-size: var(--ts-text-lg);

Utility Classes

Gebruik utility classes voor snelle styling:

<!-- Spacing --> <div class="mt-4 mb-6 px-3">...</div> <!-- Layout --> <div class="flex justify-center items-center">...</div> <!-- Grid --> <div class="ts-grid ts-grid--cols-3">...</div>

Brand Guidelines

Logo

Het Tiltshift-logo laat zien wat we doen: we helpen organisaties om net even anders te kijken. Tiltshift betekent letterlijk: de lens kantelen, spelen met perspectief. Dat is precies wat we doen bij digitale transformatie: het is geen revolutie, maar een andere invalshoek kiezen. De blokken in het logo schuiven een beetje. Ze laten zien dat verandering niet groots of ingewikkeld hoeft te zijn. Soms zit de kracht juist in kleine verschuivingen die iets in beweging zetten. Het logo past dus precies bij hoe wij werken: nuchter, stap voor stap, samen iets verschuiven waardoor de hele organisatie weer verder kan.

  • We Doen Het Echt

Merkwaarden
  • Durven Doen - iemand moet het doen en van proberen kun je leren

  • In Beweging - beweging is de natuurlijk staat van dingen en pas als er beweging is kun je bijsturen.
