Quick Start

Begin direct met het TILTSHIFT Design System in je project:

1. CSS Bestanden

Voeg alle CSS bestanden toe aan je HTML:

<link rel="stylesheet" href="tiltshift-tokens.css"> <link rel="stylesheet" href="tiltshift-base.css"> <link rel="stylesheet" href="tiltshift-atomic.css"> <link rel="stylesheet" href="tiltshift-components.css"> <link rel="stylesheet" href="tiltshift-utilities.css"> <link rel="stylesheet" href="tiltshift-themes.css">

2. Gebruik Components

Begin met atomic klassen en componenten:

<button class="ts-button ts-button--primary"> Primaire actie </button> <div class="atom-flex atom-gap-3"> <span class="molecule-tag">Tag</span> </div>

3. Style Modifiers

Pas de hele huisstijl aan met één class:

<!-- Normale stijl --> <body> <!-- Gedurfde variant --> <body class="ts-style-gedurfd"> <!-- Moderne variant --> <body class="ts-style-modern">
Tip: Combineer atomic classes voor maximum flexibiliteit. Bijvoorbeeld: atom-flex atom-gap-3 atom-items-center voor een flexibele layout.

Belangrijke concepten

  • Design Tokens: Gebruik CSS custom properties (--ts-*) voor consistente styling
  • BEM Methodology: Alle componenten gebruiken Block Element Modifier naming
  • Atomic Classes: Begin met atom-* voor basis styling, bouw op naar molecule-* en ts-* componenten
  • Responsive Design: Alle componenten zijn mobiel-eerst en volledig responsive
  • Toegankelijkheid: AtkinsonHyperlegible fonts en WCAG richtlijnen ingebouwd
