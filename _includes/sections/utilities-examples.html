Utility Classes Voorbeelden

Utility classes voor snelle styling zonder custom CSS. Alle classes gebruiken TILTSHIFT design tokens.

Spacing Utilities

Margin: .mt-1 .mb-2 .ml-3 .mr-4 .m-auto .mx-auto
Padding: .pt-1 .pb-2 .pl-3 .pr-4 .p-3
Scale: 0, xs, 1, 2, 3, 4, 5, 6 (van 0rem tot 3rem)
<div class="mt-4 mb-3 p-2">Content</div> <div class="mx-auto">Centered</div>

Layout Utilities

Display: .block .flex .grid .hidden
Flexbox: .justify-center .items-center .flex-col .gap-3
Width: .w-full .w-auto
<div class="flex justify-center items-center gap-3"> <div>Item 1</div> <div>Item 2</div> </div>

Typography Utilities

Size: .text-xs .text-sm .text-md .text-lg .text-xl .text-2xl
Align: .text-left .text-center .text-right
Font: .font-primary .font-heading .font-script
<h1 class="text-2xl font-heading text-center">Title</h1> <p class="text-md text-primary">Text</p>

Color Utilities

Text: .text-primary .text-primary-dark .text-white .text-black
Background: .bg-primary .bg-white .bg-off-white .bg-grey
<div class="bg-primary text-white p-3">Primary background</div> <span class="text-primary">Primary text</span>

Styling Utilities

Border Radius: .rounded-sm .rounded-md .rounded-lg
Shadow: .shadow-sm .shadow-md .shadow-lg
<div class="bg-white rounded-md shadow-md p-4"> Card with shadow and rounded corners </div>

Navigation Utilities

Navigation Spacing: .mt-nav .pt-nav
Automatische margin/padding voor fixed navigation (4rem mobile, 70px desktop)
<main class="pt-nav"> Content below fixed navigation </main>
