Gebruik CSS custom properties:

/* Kleuren */ color: var(--ts-color-primary); background: var(--ts-color-primary-dark); /* Spacing */ margin: var(--ts-space-3); padding: var(--ts-space-2) var(--ts-space-4); /* Typography */ font-family: var(--ts-font-primary); font-size: var(--ts-text-lg);

Atomic Design in de praktijk

Begin met atomic klassen voor maximum flexibiliteit:

<!-- Start met utilities en atoms --> <div class="a-flex a-gap-3 a-p-4"> <button class="a-button ts-button ts-button--primary"> Primaire actie </button> <span class="a-tag a-bg-primary a-text-white"> Tag </span> </div> <!-- Bouw op naar complexere molecules --> <div class="a-card a-p-4"> <h3 class="a-title-sm">Card titel</h3> <p class="a-content">Card content</p> </div> <!-- En gebruik organisms voor volledige secties --> <section class="ts-project-preview"> <!-- Complex component met alle lagen --> </section>
